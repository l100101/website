Программист микроконтроллеров

Привет! Я специализируюсь на разработке программного обеспечения для микроконтроллеров на C++. Работал с STM32, ESP32, AVR и ARM-архитектурой.

C — популярный язык программирования, удобный для работы с железом.
C++ — мощный язык для создания программ, игр и встраиваемых систем.
Embedded Systems — программирование устройств, таких как микроконтроллеры.
STM32 — микроконтроллеры, часто используемые в электронике и робототехнике.
Arduino — простая платформа для создания электронных проектов.
ESP32 — умный чип с Wi-Fi и Bluetooth для IoT-проектов.
Mbed OS — операционная система для небольших устройств.
FreeRTOS — операционная система, которая помогает программам работать в реальном времени.
Modbus — способ подключения устройств в промышленности.
UART — метод передачи данных между двумя устройствами.
I2C — шина, позволяющая подключать много устройств по двум проводам.
SPI — быстрый способ обмена данными между микросхемами.
DMX — стандарт для управления сценическим светом.
